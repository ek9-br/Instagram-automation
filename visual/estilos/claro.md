# Estilo: Claro

**Cores e acabamento** (o template define o layout; aqui é só a paleta):

- **Fundo:** off-white `#F5F7FA` / branco `#FFFFFF` (claro, arejado).
- **Texto:** grafite `#1F2933` (alto contraste sobre o claro). Apoio em slate `#3E4C59`.
- **Botão/CTA:** preenchido em teal `#14998D`, texto branco; cantos arredondados.
- **Destaques:** uma palavra/linha em teal claro `#1CD8C7` ou teal `#14998D`.
- **Overlay sobre imagem** (quando há texto sobre foto): véu **branco/off-white
  translúcido ~70–80%** atrás do bloco de texto, para legibilidade.
- **Sensação:** limpo, leve, corporativo-moderno. Máximo de 3 cores dominantes.
