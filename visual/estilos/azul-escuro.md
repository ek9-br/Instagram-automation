# Estilo: Azul escuro

**Cores e acabamento:**

- **Fundo:** azul-escuro profundo (ex.: `#0F2A43` / `#10243A`), tom escuro e sóbrio.
- **Texto:** branco `#FFFFFF` / off-white `#F5F7FA` (alto contraste sobre o escuro).
- **Botão/CTA:** preenchido em teal `#14998D` ou teal claro `#1CD8C7`, texto
  branco/escuro conforme contraste; cantos arredondados.
- **Destaques:** teal claro `#1CD8C7` para palavras-chave.
- **Overlay sobre imagem:** véu **azul-escuro translúcido ~55–70%** atrás do texto
  (escurece a foto para o texto branco ficar legível).
- **Sensação:** premium, confiável, escuro. Máximo de 3 cores dominantes.
