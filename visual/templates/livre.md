# Template: Livre

**Sem regras de layout.** Este template **não** define disposição de texto, imagem
ou botões. A composição deve seguir **fielmente a(s) imagem(ns) de referência**
anexada(s) à geração.

Match com o banco: label `Livre`.

## Como funciona
- **A imagem de referência manda.** Replique o layout, o enquadramento, a composição,
  o posicionamento dos elementos e o estilo visual da referência anexada.
- **Não imponha** nenhuma estrutura própria (sem colunas, sem ordem fixa de eyebrow/
  headline/CTA). Não invente elementos que não existam na referência.
- **Texto:** continue renderizando **exatamente** o texto fornecido da peça (sem
  inventar, traduzir ou trocar palavras), encaixando-o de forma coerente com o layout
  da referência.
- **Botão/CTA:** só se existir na copy fornecida (e/ou na referência); caso contrário,
  nenhum botão.
- **Marca:** mantenha legibilidade e a paleta da marca; o estilo selecionado é apoio,
  mas a **referência tem prioridade** quando houver conflito.

## Quando usar
- Quando você já tem uma arte/refência específica que quer reproduzir de perto,
  sem amarrar a um dos layouts fixos.
- **Requer uma imagem de referência.** Sem referência, a peça fica sem guia de
  layout — prefira um template estruturado nesse caso.
